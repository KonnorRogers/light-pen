## Installation

```bash
npm install light-pen
```

## Usage

```js
// entrypoint.js
import "light-pen"
```

```html
<!-- Your HTML file -->
<light-pen>
  <template slot="html">
    <!-- HTML -->
  </template>

  <template slot="css">
    <!-- CSS -->
  </template>

  <template slot="js">
    <!-- JS -->
  </template>
</light-pen>
```

For additional components and how to use them, check out their component pages.
